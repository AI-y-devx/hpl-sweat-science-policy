# 수동 테스트 플랜 — HPL Sweat Science v0.1.0

## 테스트 범위
- 안드로이드 앱(Google 로그인 및 클라우드 저장)
- 주요 흐름: 스플래시(2초) → 메인 → 카테고리/테스트 → 입력/저장 → 차트 보기

## 환경/디바이스 매트릭스
- 에뮬레이터: Android 14 (API 34) · ARM64 · Pixel 7 계열
- 실기기(권장 1대 이상): Android 12~14 (ARM64)
- 다크/라이트 모드, 해상도 다양성(폰/대화면)

## 사전 조건
- 릴리스 빌드(AAB→내부 테스트) 또는 디버그 APK 설치
- 네트워크 연결 가능(Wi‑Fi/LTE)

## 테스트 시나리오
1) 시작/네비게이션
   - 앱 실행 → 2초 스플래시 표시 → 메인 자동 진입
   - 뒤로가기 동작: 메인에서 뒤로가면 종료
2) 카테고리/테스트 이동
   - 생리/퍼포먼스/동작/인지 메뉴 진입·복귀 정상
3) 입력/저장
   - 각 테스트 화면에서 유효값/경계값 입력 후 저장
   - 비정상 값(음수/과도값) 처리(경고/거부) 확인
4) 차트
   - Syncfusion 차트 렌더링(툴팁/범례 동작)
   - 다크/라이트에서 가독성 확인
5) 오프라인 동작
   - 네트워크 불가 시 로그인/클라우드 저장 동작(에러 처리/재시도) 확인
6) 회전/멀티태스킹
   - 가로/세로 전환 시 레이아웃 정상
   - 백그라운드→포그라운드 재개 문제 없음
7) 성능
   - 첫 화면 표시(스플래시 제외) 2초 내
   - 스크롤/차트 인터랙션 중 렉/프리즈 없음
8) 안정성
   - 30분 탐색 중 크래시/ANR 0건

## 로그/리포트 수집
- 크래시/예외: `adb logcat | egrep -i "flutter|dart|fatal|exception|anr"`
- 성능(간이): Profile 모드 실행, 잔상/프레임 드랍 수동 체감 기록
- 스크린샷/영상 첨부

## 합격 기준
- 크래시·ANR 0, 막다른 네비게이션 0
- 주요 시나리오 100% 통과
- 프리런치 리포트 차단 이슈 0
