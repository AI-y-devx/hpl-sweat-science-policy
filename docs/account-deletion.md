# 계정 삭제 안내 (HPL Sweat Science)

본 안내는 **HPL Sweat Science** 앱의 계정 삭제(탈퇴) 방법 및 삭제/보관되는 데이터 범위를 설명합니다.

## 1. 계정 삭제(탈퇴) 방법
1) 앱 실행
2) 로그인 후 **프로필(Profile)** 화면으로 이동
3) **계정 삭제** 버튼 선택
4) 안내 문구 확인 후 삭제 진행

보안상 이유로, 기기/계정 상태에 따라 **최근 로그인(재인증)**이 필요할 수 있습니다.

## 2. 삭제되는 데이터
계정 삭제 시 다음 정보는 삭제됩니다.

- 사용자 프로필 정보(`users/{uid}`):
  - 이메일
  - 표시 이름
  - 이름, 생년월일, 신장, 체중 등 프로필 입력값
- 앱 내 로컬 저장 데이터(설정/캐시 등)
- Firebase Authentication 사용자 계정(로그인 계정)

## 3. 보관(익명화)되는 데이터
계정 삭제 시 다음 정보는 서비스 품질 향상 및 연구/개선 목적을 위해 **개인 식별정보를 제거한 익명 형태로 보관**될 수 있으며, 일반적으로 삭제되지 않습니다.

- 측정/기록 데이터(`measurements`)
  - 계정과 연결된 소유자 정보는 익명화 처리됩니다(예: `ownerUid`가 익명 값으로 변경).

익명화된 데이터는 특정 개인과의 연결고리가 제거되므로, **익명화 이후에는 특정 개인의 기록만을 선별하여 삭제하는 것이 제한**될 수 있습니다.
